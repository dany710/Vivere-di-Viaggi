# Vivere di Viaggi - Sito Responsive

Questo repository contiene un sito web statico per "Vivere di Viaggi", un partner indipendente inCruises. Il sito è ottimizzato per dispositivi mobili e pronto per la pubblicazione su GitHub Pages.

## Caratteristiche
- **Mobile Friendly**: Navigazione responsive con menu hamburger.
- **Integrazione WhatsApp**: Moduli di contatto e pulsanti configurati per la lead generation tramite WhatsApp.
- **Design Moderno**: Realizzato con Tailwind CSS.

## Struttura del Progetto
- `index.html`: Homepage principale del sito.
- `chi-siamo.html`: Informazioni sulla missione e la visione.
- `destinazioni.html`: Catalogo delle destinazioni crocieristiche.
- `membership.html`: Dettagli sui piani e le opportunità.
- `contatti.html`: Modulo di contatto e informazioni partner.
- `landing-page.html`: Pagina di vendita ottimizzata.

## Pubblicazione su GitHub Pages
Il sito è configurato per essere servito direttamente dalla root del branch principale. La presenza del file `.nojekyll` assicura che tutti i file vengano serviti correttamente senza elaborazioni lato server.
